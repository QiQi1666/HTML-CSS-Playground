Lists

This is how you create an unordered list:

  • Add a "ul" element
  • Add each item in its own "li" element
  • They don't need to be in a particular order
  • You can also nest a "p" element in "li" elements

This is how you create an ordered list:

  1. The new "ol" element is wrapping everything instead
  2. But the list item elements stay the same
  3. Note how the numbers increment on their own
  4. You should be noticing things in this exact order
